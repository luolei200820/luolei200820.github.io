# 反射

# Class类

每个类是由JVM动态加载的，当第一次读到一种类时，就创建一个Class类的实例。注意这里的Class类是一个名字叫做Class的类。

获取Class类实例的三种方式

  1. 通过类名：Class c1 = String.class;
  2. 通过实例变量：String s1 = ""; Class c2 = s1.getClass();
  3. 通过完整类名：Class c3 = Class.forName("java.lang.String");

这三种方式获取到的都是同一个实例：c1==c2==c3;因为JVM只对加载的每一个类创建一个Class实例来表示它的类型。

# 访问和设置字段

  • Field getField(name)：根据字段名获取某个public的field（包括父类）
  • Field getDeclaredField(name)：根据字段名获取当前类的某个field（不包括父类）
  • Field[] getFields()：获取所有public的field（包括父类）
  • Field[] getDeclaredFields()：获取当前类的所有field（不包括父类）

一个Field对象包含了一个字段的所有信息：

  • getName()：返回字段名称，例如，"name"；
  • getType()：返回字段类型，也是一个Class实例，例如，String.class；
  • getModifiers()：返回字段的修饰符，它是一个int，不同的bit表示不同的含义。
public class Test {
    public static void main(String[] args) throws Exception{
        Object p = new Person("xiao ming");
        Class c = p.getClass();
        Field f = c.getDeclaredField("name");
        f.setAccessible(true); // 设置字段的访问权限

        // 设置
        f.set(p,"xiao hong");

        // 获取
        Object value = f.get(p);
        System.out.println(value);
    }
}

class Person {
    private String name;

    public Person(String name) {
        this.name = name;
    }
}

# 调用方法

public class Test {
    public static void main(String[] args) throws Exception {
        String s = "hello world";
        // 获取Method对象，substring两个参数的方法，参数为int
        Method m = s.getClass().getMethod("substring", int.class, int.class);
        // 在s上调用该方法
        String r = (String) m.invoke(s, 1, 6);
        System.out.println(r);
    }
}

# 调用静态方法

public class Test {
    public static void main(String[] args) throws Exception {
        Method m = Integer.class.getMethod("parseInt", String.class);
        // 因为静态方法无需指定实例对象，因此第一个参数为null
        int n = (int) m.invoke(null, "123");
        System.out.println(n);
    }
}

# 调用非public方法

和获取和访问非public字段类似，可以通过Method.setAccessible(true)设置允许访问。

# 多态

使用反射调用方法仍然遵循多态原则：即总是调用实际类型的重写方法。

public class Main {
    public static void main(String[] args) throws Exception {
        // 获取Person的hello方法:
        Method h = Person.class.getMethod("hello");
        // 对Student实例调用hello方法:
        h.invoke(new Student());
        // 实际输出的是Student:hello
    }
}

class Person {
    public void hello() {
        System.out.println("Person:hello");
    }
}

class Student extends Person {
    public void hello() {
        System.out.println("Student:hello");
    }
}

# 调用构造方法

Person p = Person.class.newInstance();

缺点是只能调用该类的public无参构造方法。为了调用任意的构造方法，Java API提供了Constructor对象。

// 获取通过int类型创建Integer类型的构造函数
Constructor c1 = Integer.class.getConstructor(int.class);
Integer n1 = (Integer) c1.newInstance(123);
System.out.println(n1);

// 获取通过String类型创建Integer类型的构造函数
Constructor c2 = Integer.class.getConstructor(String.class);
Integer n2 = (Integer) c2.newInstance("456");
System.out.println(n2);

同样调用非public构造方法时要通过setAccessible(true)设置允许访问。

Class i = Integer.class;
Class is = i.getSuperclass();
System.out.println(is.getName());

# 获取实现的接口

public class Main {
    public static void main(String[] args) throws Exception {
        Class s = Integer.class;
        Class[] is = s.getInterfaces();
        for (Class i : is) {
            System.out.println(i);
        }
    }
}

运行上述代码可知，Integer实现的接口有：

  • java.lang.Comparable
  • java.lang.constant.Constable
  • java.lang.constant.ConstantDesc

要特别注意：getInterfaces()只返回当前类直接实现的接口类型，并不包括其父类实现的接口类型：

接口的getClass()为null，要获取父接口要用getInterfaces()

# 动态代理

接口不能直接实例化，只能借助实现类创建实例对象并向上转型才可以。

JDK提供的动态创建接口对象的方式，叫做动态代理。

public class Main {
    public static void main(String[] args) {
        InvocationHandler handler = new InvocationHandler() {
            @Override
            public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
                System.out.println(method);
                if (method.getName().equals("morning")) {
                    System.out.println("Good morning, " + args[0]);
                }
                return null;
            }
        };
        Hello hello = (Hello) Proxy.newProxyInstance(
            Hello.class.getClassLoader(), // 传入ClassLoader
            new Class[] { Hello.class }, // 传入要实现的接口
            handler); // 传入处理调用方法的InvocationHandler
        hello.morning("Bob");
    }
}

interface Hello {
    void morning(String name);
}
