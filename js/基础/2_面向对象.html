# 面向对象

# 使用工厂方法创建对象

function person(name, age) {
  var obj = new Object()

  obj.name = name
  obj.age = age

  return obj
}

person('luolei', 18)

# 构造函数

// 函数名首字母大写以区分普通函数
function Person(name, age) {
  this.name = name
  this.age = age
}

var per = new Person('sun', 18)

# 原型

定义在构造函数对象的 prototype 属性上，可以被所有实例对象共享

使用案例：如果将相同方法都写构造函数里，则每次调用构造函数时都会将此方法实例化，浪费内存，如果将其定义在原型上则可以共享

Person.prototype.hello = function () {
  alert('hello')
}

# 继承

继承可以分为两步实现：第一步是在子类的构造函数中调用父类的构造函数

function Sub(value) {
  Super.call(this)
  this.prop = value
}

第二步，是让子类的原型指向父类的原型，这样子类的就可以继承父类原型

WARNING

注意不要直接将Sub.prototype赋值Super.prototype，否则对子类原型的修改会连同父类的原型一起修改。

Sub.prototype = Object.create(Super.prototype)
Sub.prototype.constructor = Sub // 需要修改子类原型的constructor属性，指向自身的构造函数

# 多重继承

function M1() {
  this.hello = 'hello'
}
function M2() {
  this.world = 'world'
}

// 继承M1
S.prototype = Object.create(M1.prototype)
// 继承链上加入M2，将M2原型对象上自身具有的可遍历属性拷贝到S的原型对象
Object.assign(S.prototype, M2.prototype)

// 指定构造函数
S.prototype.constructor = S

# Object的相关方法

Object上有很多方法，处理面向对象的有以下方法：

  • Object.getPrototypeOf()
  • Object.setPrototypeOf()
  • Object.create()
  • Object.prototype.isPrototypeOf()
  • Object.getOwnPropertyNames()
  • Object.prototype.hasOwnProperty()

# 改变函数内部的 this

  • call(thisValue,arg1,arg2,...)，改变函数内部的 this，同时执行函数

  • apply(thisValue,[arg1,arg2,...])，改变函数内部的 this，同时执行函数

    有趣的应用：找出数组最大元素

    var a = [10, 2, 4, 5, 6]
Math.max.apply(null, a)

  • bind(thisValue)，绑定函数内部的 this 的值为一个对象，然后将这个函数返回

# 遍历对象属性

  • Object.keys()：返回对象自身可枚举属性组成数组
  • for ... in：遍历对象的所有可枚举属性，包括继承的可枚举属性，可以用'key' in Object来判断一个属性是否在对象上（不管是否是继承的，或者是不可遍历的）
  • Object.getOwnPropertyNames()：获取对象自身的所有属性，包括不可枚举属性
  • JSON.stringify()：将对象以 JSON 格式输出，排除不可枚举属性
