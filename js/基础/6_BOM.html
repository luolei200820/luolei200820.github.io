# BOM

浏览器对象模型，BOM 可以使我们用 JS 操作浏览器，在 BOM 中为我们提供了一组对象，用来完成对浏览器的操作

  • Window
    代表的是整个浏览器的窗口，同时 window 也是网页中的全局对象
  • Navigator
    代表当前浏览器的信息，通过该对象可以识别不同的浏览器
    由于历史原因，Navigator 对象中的大部分属性已经无法帮助识别浏览器了
    一般使用 userAgent 来判断浏览器的信息，不同的浏览器会有不同的 userAgent
    • 火狐含有 firfox，Chrome 还有 chrome，IE10 记以前有 MSIE，IE11 与 IE 相关标识以去除
    • 可以通过判断浏览器中是否有特有的属性来进一步判断，比如 IE 中有 currentStyle、attachEvent、ActiveXObject
  • Location
    代表当前浏览器的地址栏信息，通过它可以获取地址栏信息，或者操作浏览器跳转页面
  • History
    代表浏览器的历史记录，可以通过该对象来操作浏览器的历史记录
    由于隐私的原因，该对象不能获取到具体的历史记录，只能操作浏览器向前或者向后，而且该操作只能在当前访问有效
  • Screen
    代表用户的屏幕的信息，通过该对象可以获取到用户的显示器的相关信息

这些 BOM 对象在浏览器中都是作为 window 对象的属性保存的

  • length

# 方法

  • back()
    可以回退到上一个页面，和浏览器的回退按钮一样
  • forward()
    前进
  • go()
    可以用来跳转到指定的页面
    他需要一个整数作为参数
    • 正数表示前进
    • 负数表示后退
  • assign()
    和直接给 location 赋值一样
  • reload()
    重新加载页面
    • 可以传一个布尔值 ture 来强制刷新整个页面
  • replace() 使用一个新的页面来替换当前页面
    • 不会生成历史记录
