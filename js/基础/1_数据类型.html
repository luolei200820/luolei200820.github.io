# 数据类型

# 基本类型

有五种基本类型： string,number,boolean,null,undefined

# null 与 undefined 的区别

  • undefined
    变量定义了，但是没有赋值（如果没有定义则会报错 Uncaught ReferenceError: a is not defined）
  • null
    变量定义了，并且赋值为 null

# 什么时候给变量赋值为 null 呢？

  • 初始赋值，表明此变量将要赋值为对象
  • 结束时，将对象作为垃圾对象（被垃圾回收器回收）

# 不加分号

不加分号的语法风格需要注意两点

  • 小括号开头的前一条语句需要加分号
  • 中括号开头的前一条语句需要加分号

解决办法，在行首加分号

# 栈内存和堆内存

变量名和变量的值存储在栈内存中
基本类型数据的值存储在栈内存中，而对象类型的数据存储在堆内存中
当访问对象时，先到栈内存中获取对象的地址值，再到堆内存中寻找对应的对象

# 对象类型

有三种对象类型： Object、Array、Function

所有对象都是 Object 类型的实例，包括数组和函数

[] instanceof Array // true
[] instanceof Object // true

var fun = function(){}
fun instanceof Function // true
fun instanceof Object // true

# this

  1. 以函数形式调用时，this 是 window
  2. 以方法调用时，this 是调用方法的对象
  3. 以构造函数调用时，this 是新创建的那个对象
  4. 使用 call 和 apply 调用时，this 是指定的那个对象

# call()和 apply()

这两个方法都是函数对象的方法，需要通过函数对象来调用。
当函数调用 call()和 apply()都会调用函数执行。
可以将一个对象指定为第一个参数，此时这个对象将会成为函数执行时的上下文对象（this）。
实参可以在 call()方法的上下文对象之后依次传递，而 apply()方法需要将实参封装到一个数组中传递。

fun.call(obj, 2, 3)
fun.apply(obj, [2, 3])

# arguments

在调用函数的时候，浏览器每次都会传递两个隐含的参数

  1. 函数的上下文 this
  2. 封装实参的对象 arguments
    • arguments 是一个类数组的对象，它可以通过索引来操作数据，也可以获取长度
    • 在调用函数时，我们所传递的实参都在 arguments 中保存。可以通过索引来使用实参，只不过比较麻烦：arguments[1]
    • callee属性，这个属性对应于一个函数对象，就是当前正在指向的函数的对象

# 数组属性

  • length —— 数组长度

# 数组方法

  • push() —— 向数组的末尾添加一个或更多元素，并返回新的长度
    影响原数组
  • pop() —— 删除并返回数组的最后一个元素
    影响原数组
  • unshift() —— 向数组的开头添加一个或更多元素，并返回新的长度
    影响原数组
  • shift() —— 删除并返回数组的第一个元素
    影响原数组
  • slice() —— 从某个已有的数组返回选定的元素
    不影响原数组
    第一个参数是开始的索引位置
    第二个参数是索引结束的位置（不包含）
    可以省略第二个参数，这么做会截取到数组的最后
  • splice() —— 删除元素，并向数组添加新元素
    影响原数组
    第一个参数是开始的索引位置
    第二个参数是删除元素的个数
    第三个参数及以后表示要在开始索引位置之前插入的新元素
    返回被删除的元素
  • forEach() —— 遍历元素
    第一个参数是回调函数，回调的第一个参数是元素本身，第二个参数是元素的索引，第三个参数是数组对象
    第二个参数是回调函数里的 this 的值
  • concat() —— 连接两个或多个数组，并将新数组返回
    不影响原数组
  • join() —— 该方法可以将数组转换为字符串并返回
    不影响原数组
    参数为连接符，默认为逗号 ','
  • reverse() —— 反转数组元素
    影响原数组
  • sort() —— 对数组元素排序
    影响原数组
    默认按 Unicode 排序
    可以传入一个回调函数来指定排序的方式
    浏览器会分别使用数组中的元素作为实参去调用回调函数
    回调函数有两个形参(a,b)，数组中 a 一定在 b 前边
    回调函数的返回值决定元素的顺序，>0 前面的大交换，=0 相等不变，<0 前面的小不变
