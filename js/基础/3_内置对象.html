# String 对象

在底层，字符串是以字符数组的形式保存的（类似 C 语言），因此也可以通过索引访问单个字符

# 属性

  • length —— 字符串长度

# 方法

基本不影响原字符串

  • chatAt() —— 返回指定位置的字符
  • charCodeAt() —— 返回指定位置字符的 Unicode 编码
  • fromCharCode() —— 根据字符编码返回字符
  • concat() —— 连接
  • indexOf() —— 检索一个字符串中是否含有指定内容
    • 如果该字符串中含有该内容，则会返回其第一次出现的索引，如果没有找到则返回-1
    • 可以指定第二个参数，指定开始查找的位置
  • lastIndexOf() —— 用法和 indexOf 相同，不同的是从后往前找
  • slice() —— 可以从字符串中截取指定内容
    • 不会影响原字符串
    • 参数
      1. 开始位置的索引
      2. 结束位置的索引，省略则表示截取到末尾
  • substring() —— 与 slice 类似，只不过不接受结束位置索引为负值，自动调整参数位置如果第一个大于第二个的话
  • split() —— 把字符串分割成数组
    • 参数：需要一个字符串作为参数，将根据该字符串拆分数组
  • toLowerCase() 转小写
  • toUpperCase() 转大写

# 正则表达式对象

# 两种创建方式

正则表达式对象：RegExp()
正则表达式字面量 /a/

# 正则表达式语法

# 修饰符

写在字面量最后

  • i —— 大小写不敏感
  • g —— 执行全局匹配，查找所有匹配而非在第一个匹配后停止
  • m —— 执行多行匹配

# 元字符

写在双斜杠里以表示某种字符

  • . —— 表示任意字符（如果要匹配点，则要使用反斜杠\转义）
  • \w —— 任意字母、数字、下划线（以下可以简单成对记忆）
  • \W —— 除了字母、数字、下划线
  • \d —— 任意数字
  • \D —— 除了任意数字
  • \s —— 空格
  • \S —— 除了空格
  • \b —— 单词边界（如/\bchild\b/匹配单词 child，child 前后需要是空或空格）
  • \B —— 除了单词边界

# 普通用法

  • /a/ —— 检查字符串中是否含有 a

# 小括号

  • 标记一个子表达式
  • /(ab){3}/表示 ab 连续出现 3 次

# 中括号

  • /[abcd]/ —— 匹配中括号中的任意单个字符
  • /[^a]/ —— 中括号中有^表示匹配除了括号中的内容
  • /[a-z]/——表示字符范围

# 竖杠

  • /ab|cd|ae/ —— 表示字符串中含有 ab 或者 cd 或者 ae

# 量词

  • 量词只对前边一个内容起作用，/(ab){3}/表示 ab 连续出现 3 次
  • {n} 正好出现 n 次
  • {m,n} 出现 m~n 次
  • {m,} 出现 m 次以上
  • + 至少一个
  • * 零个或多个
  • ? 零个或一个

# 开头结尾匹配

  • /^a/ —— 检查一个字符串是否以 a 开头
  • /a$/ —— 检查一个字符串是否以 a 结尾
  • 如果在表达式中同时使用^和$，则要求字符串必须完全符合正则表达式

# 举例

  1. 手机号码

    /*
1. 以1开头
2. 第二位3-9任意数字
3. 三位以后任意数字9个
*/

var reg = /^1[3-9][0-9]{9}$/

# 和字符串方法结合

  • split() —— 将字符串按给出的正则表达式拆分成数组（自动全局匹配）

  • search() —— 可以搜索字符串中是否含有指定内容

    • 如果搜索到指定内容，则会返回第一次出现的索引（不能全局匹配）
    • 如果没有搜索到则返回-1
    • 可以接受一个正则表达式作为参数，然后根据正则表达式去检索字符串

  • match()

    • 若正则表达式不是全局匹配的，则返回一个类数组，包含匹配的字符串、索引、输入
    • 若是全局匹配则，则返回一个匹配正则的字符串数组

  • replace()

    去掉开头的空格

    var str = '      he  llo     '
str = str.replace(/^\s+/, '')

    ​ 去掉开头和结尾的空格

    var str = '      he  llo     '
str = str.replace(/^\s+|\s+$/g, '')

# Date 对象

在 JS 中使用 Date 对象来表示一个时间

如果直接使用 Date 构造函数来创建对象，则会封装为当前代码执行的时间

var d = new Date()
console.log(d)

//输出：Sun Sep 12 2021 20:33:43 GMT+0800 (中国标准时间)

如果想要创建指定的时间，则需要传递一个字符串参数给构造函数
格式： 月/日/年 时:分:秒

var d2 = new Date('12/03/2011 11:10:30')
console.log(d2)

// 输出：Sat Dec 03 2011 11:10:30 GMT+0800 (中国标准时间)

# 对象方法

  • getDate() —— 返回一个月中的某一天，1~31
  • getDay() —— 返回一周中的某一天，0：星期天 1：星期一
  • getMonth() —— 返回月份，0：一月，11：十二月
  • getFullYear() —— 获取年份
  • getTime() —— 获取日期对象的时间戳
    • 时间戳：指的是从格林威治时间 1970 年 1 月 1 日 0 时 0 分 0 秒到当前日期所花费的毫秒数
    • 1 秒 = 1000 毫秒

# Math 对象

# 属性

  • E —— 自然对数 e
  • PI —— 圆周率
  • ...

# 方法

  • abs() —— 绝对值

  • round() —— 四舍五入

  • ceil() —— 上舍入

  • floor() —— 下舍入

  • max() —— 返回 x,y,z,...的最大值

  • min()

  • random() —— 返回 0~1 之间的随机数

    • 返回 0~x 之间的随机数
    Math.round(Math.random() * x)
    • 返回 x~y 之间的随机数
    Math.round(Math.random() * (y - x) + x)

  • pow() —— 返回 x 的 y 次幂

  • sqrt() —— 开方
