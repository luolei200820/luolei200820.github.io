# 查询文档

# 选择集合中所有的文档

db.inventory.find({})

# 指定相等的条件

db.inventory.find({ status: 'D' })

# 使用查询操作符指定条件

以下的例子查询status等于"A"或者"D"

db.inventory.find({ status: { $in: ['A', 'D'] } })

TIP

尽管也可以使用$or操作符，但在对同一个字段时使用$in

# 指定AND条件

一个复合查询可以对集合中文档的多个字段指定多个条件。隐含地，一个逻辑AND连词将复合查询的子句连接起来，因此可以选中集合中满足所有条件的文档。

下面的例子获取inventory集合中所有status等于"A"并且qty小于($lt)30 的文档

db.inventory.find({ status: 'A', qty: { $lt: 30 } })

# 指定OR条件

使用$or操作符，你可以指定一个使用逻辑OR连词将复合查询的子句连接起来的复合查询，因此查询选中集合中哪些满足至少一个条件的文档。

下面的例子，获取集合中所有那些status等于A或者qty小于($lt)30 的文档

db.inventory.find({ $or: [{ status: 'A' }, { qty: { $lt: 30 } }] })

# 指定AND和OR条件

In the following example, the compound query document selects all documents in the collection where the status equals "A" and either qty is less than ($lt) 30 or item starts with the character p:

下面的例子，复合查询文档选择集合中那些status等于A并且qty小于($lt)30 或者item以字符p开头的所有文档

db.inventory.find({
  status: 'A',
  $or: [{ qty: { $lt: 30 } }, { item: /^p/ }],
})

TIP

MongoDB 支持正则表达式$regex查询来执行字符串模式匹配
